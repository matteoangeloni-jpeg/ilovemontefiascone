## Issues Resolved

- Replaced mojibake-heavy DE core with UTF-8 German pages.
- Removed public hreflang from German pages.
- Added self-referential future canonicals on non-www `https://www.ilovemontefiascone.com/de/...` URLs.
- Added visible breadcrumbs and matching `BreadcrumbList` JSON-LD.
- Added visible FAQ and matching `FAQPage` JSON-LD on all core pages.
- Added `inLanguage: de` to structured data.
- Built DE-to-DE internal linking only.
- Avoided invented prices, opening hours, exact travel times, hotel names, restaurant names and event dates.

## Residual Issues

- P1: German copy should still receive a final native editorial read before publication.
- P1: Some pages are intentionally concise and should be enriched only with verified information later.
- P2: Images are shared generic Montefiascone assets; page-specific imagery can be added before launch.
- P2: No public language switcher, sitemap or hreflang should be added until publication conditions are met.

## QA Checklist for Future Publication

- 0 mojibake.
- 0 Italian or English residue outside proper names and technical metadata.
- All titles under 60 characters.
- All meta descriptions between 120 and 155 characters.
- Exactly one H1 per page.
