# DE Glossary - IT / EN / DE

Updated: 2026-06-26

| Italiano | English | Deutsch | Usage rule |
| --- | --- | --- | --- |
| centro storico | old town / historic centre | Altstadt / historisches Zentrum | Use "Altstadt" in travel copy; "historisches Zentrum" for formal context. |
| Lago di Bolsena | Lake Bolsena | Bolsenasee | Translate in German as "Bolsenasee"; mention Italian name only when useful. |
| Rocca dei Papi | Fortress of the Popes | Rocca dei Papi | Keep Italian name; explain as Aussichtspunkt/Festungsanlage on first mention. |
| Cattedrale | Cathedral | Kathedrale | Translate generic term; keep "Santa Margherita" as proper name. |
| Basilica | Basilica | Basilika | Translate generic term; keep "San Flaviano" as proper name. |
| eventi estivi | summer events | Sommerveranstaltungen | Use for seasonal hub pages. |
| cantina | winery / cellar | Weingut / Kellerei | Use "Weingut" for producers, "Kellerei" for cellar context. |
| vino | wine | Wein | Translate always. |
| enogastronomia | food and wine | Kulinarik / Wein und Küche | Prefer natural German phrase over literal compound. |
| cosa vedere | things to see | Sehenswürdigkeiten | Use as navigation label and H1 theme. |
| dove mangiare | where to eat | Essen in / wo essen | Use "Essen in Montefiascone" for page title. |
| itinerario | itinerary | Route / Reiseplan | Use "Route" for practical touring; "Reiseplan" for planning. |
| guida | guide | Reiseführer | Use "Reiseführer" for hub pages. |
| borgo | hill town / village | Ort / historischer Ort | Avoid overusing "Borgo" in German unless explained. |
| Tuscia | Tuscia | Tuscia | Keep as regional-cultural name; explain as northern Lazio near Viterbo. |
| Lazio | Lazio | Latium | Translate to "Latium" in German text. |
| Belvedere | viewpoint | Aussichtspunkt | Translate in ordinary German. |
| pellegrinaggio | pilgrimage | Pilgerreise / Pilgerweg | Use "Pilgerweg" for Via Francigena context. |
| Via Francigena | Via Francigena | Via Francigena | Keep Italian route name; explain as historischer Pilgerweg. |
| Fiera del Vino | Wine Fair | Fiera del Vino | Keep Italian event name; add "Weinfest" only as explanatory wording in body text. |
| Cronoscalata | hill climb | Bergrennen / Cronoscalata | Use "Bergrennen" for German readability; keep the official Italian name when referring to the event. |
